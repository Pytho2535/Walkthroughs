# Walkthroughs
## 🧠 O Repozytorium

To repozytorium zawiera moje walkthroughy z platformy LetsDefend, gdzie rozwiązuję scenariusze bezpieczeństwa IT. Każdy plik zawiera krok po kroku analizę zagrożeń, logów i podejścia do incydentów.

## 🧭 Jak korzystać

Każdy plik `.md` to osobny walkthrough konkretnego zadania. Możesz przejść przez analizę krok po kroku razem ze screenami i krótkimi wyjaśnieniami. Używam jasnego schematu: pytanie → analiza → log → odpowiedź.
